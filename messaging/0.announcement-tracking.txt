The W3C Accessible Platform Architectures Working Group Maturity Model Task Force has published the Accessibility Maturity Model Group Note. The document describes an overall framework for establishing a robust ICT accessibility program by identifying and tracking areas for improvement. This version is the first public Note published by the Accessible Platform Architectures Working Group Maturity Model Task Force, developed with the community, as a version 1.0. The approaches identified in this Note are intended to influence organizational programs and processes to advance accessibility with a holistic focus.
